# RootReveal
一款安卓环境检测工具，专注于 Root / 系统修改痕迹检测

> 项目官网：https://rootreveal.netlify.app/

---

## 🔍 项目介绍
RootReveal 是一款面向安卓设备的环境检测工具，主要用于检测设备的系统修改痕迹，包括但不限于 Root 环境、隐藏模块、系统篡改特征等。

工具仅提供**环境检测功能**，不提供任何破解、作弊、绕过检测的功能，也不提供任何违反平台规则的工具或教程。

---

## ✨ 核心检测能力
- 基础 Root 环境特征检测（su 二进制、SELinux 状态、系统属性）
- 常见 Root 管理工具痕迹识别（Magisk / KernelSU / APatch 相关特征）
- 进程与内存特征扫描，识别常见隐藏/注入模块
- 文件系统关键路径特征检测，识别系统修改痕迹
- Bootloader 解锁与系统完整性基础检测

---

## 📱 支持平台
- 系统要求：Android 8.0 及以上
- 设备架构：arm64-v8a（主流安卓机型通用）
- 适配情况：针对国内主流机型与系统（ColorOS / MIUI / HyperOS 等）做了适配优化
